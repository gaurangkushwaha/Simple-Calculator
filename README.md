# Simple Calculator Program in Python

## 📌 Overview
This is a beginner-friendly Python project that performs basic arithmetic operations using conditional statements.

The calculator takes two numbers and an operator from the user and performs:
- Addition
- Subtraction
- Multiplication
- Division

It also handles:
- Invalid operations
- Division by zero errors


## 🚀 Features
- User input handling
- Arithmetic operations
- Nested `if-else` conditions
- Error handling
- Beginner-friendly code structure

---

## 🛠️ Technologies Used
- Python 3
